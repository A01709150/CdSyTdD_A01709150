Preguntas del Laboratorio 6:

¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Es importartante hacer esto para que no haiga problema a la hora de enviar los datos al servido.
De esta manera el usuario se puede corregir si es que hubo una equivocación, como por ejemplo si ingreso
su contraseña mal.

2) ¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Ir a navegador y desactivar javascript

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Porque igualmente sirve para tener mas seguridad. Algo es mejor que nada.
